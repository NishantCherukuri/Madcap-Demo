Multiple Menus

Multiple menu proxies can be added to a topic to support different navigation needs. On this page, one menu provides navigation for child topics in the TOC, while another displays the page headings to help readers move through the content more easily.

Implementation Steps

Add a Menu Proxy

  1. Place your cursor in a template page. You may need to press the down arrow on your keyboard and then press ENTER at the correct location. Then select Insert > Proxy > Insert Menu Proxy.

  2. After inserting a Menu proxy, you can edit it to display a specific TOC, Browse Sequence, or make it Context Sensitive. To do this, open the topic or template page where the proxy is inserted, right-click on it, and select Edit Menu Proxy.

Add Another Menu Proxy

  1. Place your cursor in the template page, below the first menu proxy. Then select Insert > Proxy > Insert Menu Proxy.

  2. After inserting a Menu proxy, edit the menu proxy so that it has different settings than a TOC such as to display headings on the page. To do this, open the topic or template page where the proxy is inserted, right-click on it, and select Edit Menu Proxy.

Copy + Paste Method

Alternatively you can copy and paste the code below into the template page via the internal text editor. Then edit the menu proxies in the XML editor.

Copy 
<MadCap:menuProxy style="mc-toc-depth: 1;mc-context-sensitive: False;mc-include-parent: True;mc-include-siblings: True;mc-include-children: True;"  />
                    
<MadCap:menuProxy mc-linked-toc="$topicHeadings"  style="mc-context-sensitive: True;mc-include-parent: True;mc-include-siblings: True;mc-include-children: True;"  />

For more information on menu proxies, see our online help.
