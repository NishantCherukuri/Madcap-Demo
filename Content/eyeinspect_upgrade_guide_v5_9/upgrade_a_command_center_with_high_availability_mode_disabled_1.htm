Upgrade a Command Center with High Availability Mode Disabled

 

Note: Ensure that the eyeInspect appliances containing the Command Centers are upgraded to Ubuntu 22.04.4 before proceeding.

  1. Log in to the Command Center using SSH as the silentdefense user.
  2. Transfer to the machine in the /home/silentdefense/ directory the following file: 
    eyeinspect_commandcenter_5.5.1_update.run
  3. Verify that the required files are present by issuing the following command to list the contents of the current directory: 
    ls -alh
  4. Ensure the updaters are marked executable by issuing the command: 
    chmod a+x *.run
  5. Upgrade the Command Center by issuing the command: 
    sudo ./eyeinspect_commandcenter_5.5.1_update.run
  6. Once the Command Center is upgraded, reboot the system by issuing the command: 
    sudo reboot
Verify that the upgrade is successful by logging in the server via SSH and looking at the welcome message - it lists the installed components and their versions. This check can also be performed manually by issuing the command: 
dpkg -l| grep silentdefense-cc
